02
Obtener API Keys
OpenAI + ElevenLabs — cuentas gratuitas disponibles

OpenAI (ChatGPT): Ve a platform.openai.com → API Keys → Create new secret key. Los primeros $5 USD son gratis con cuenta nueva.

ElevenLabs: Ve a elevenlabs.io → Sign up gratuito → Profile → API Key. El plan gratis incluye 10,000 caracteres/mes (~15 minutos de audio).

⚠️ En producción, nunca expongas tus API keys en el frontend. Usa un backend (Node.js, Python) como proxy. Esta demo es solo para aprendizaje local.
03
Integración con ChatGPT
OpenAI Chat Completions API

Se usa el endpoint /v1/chat/completions. Mandas un array de mensajes con roles: system (personalidad), user (tú) y assistant (respuestas anteriores).

const callChatGPT = async (messages) => {
  const res = await fetch("https://api.openai.com/v1/chat/completions", {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${OPENAI_KEY}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      model: "gpt-4o-mini",       // modelo a usar
      messages: messages,          // historial completo
      max_tokens: 200,             // límite de respuesta
      temperature: 0.7            // creatividad (0-1)
    })
  });
  const data = await res.json();
  return data.choices[0].message.content;
};

El array messages mantiene toda la conversación. Así ChatGPT recuerda el contexto. Limítalo a los últimos 10 mensajes para no gastar tokens de más.

04
Síntesis de voz con ElevenLabs
Text-to-Speech API — voces ultra-realistas

ElevenLabs recibe texto y retorna audio binario (mp3). Se usa el endpoint /v1/text-to-speech/{voice_id}.

const speakWithElevenLabs = async (text) => {
  const res = await fetch(
    `https://api.elevenlabs.io/v1/text-to-speech/${VOICE_ID}`,
    {
      method: "POST",
      headers: {
        "xi-api-key": ELEVENLABS_KEY,
        "Content-Type": "application/json"
      },
      body: JSON.stringify({
        text: text,
        model_id: "eleven_multilingual_v2", // soporta español
        voice_settings: {
          stability: 0.5,          // consistencia de voz
          similarity_boost: 0.75  // fidelidad al clon
        }
      })
    }
  );
  const audioBlob = await res.blob();
  const audioUrl = URL.createObjectURL(audioBlob);
  new Audio(audioUrl).play();
};
🌍 Usa el modelo eleven_multilingual_v2 para español. Las voces en inglés también funcionan bien con texto en español.
05
Publicar en GitHub Pages
Deploy gratuito en menos de 5 minutos

GitHub Pages sirve archivos estáticos gratis. Solo necesitas un repositorio público.

# 1. Crea un repo en github.com (ej: "voice-chatbot")

# 2. Sube tu index.html
git init
git add index.html
git commit -m "feat: add voice chatbot"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/voice-chatbot.git
git push -u origin main

# 3. En GitHub: Settings → Pages → Source: main / root
# Tu URL será: https://TU_USUARIO.github.io/voice-chatbot
🔒 Para una versión segura, crea un backend mínimo en Vercel + Node.js o Supabase Edge Functions que actúe como proxy entre el frontend y las APIs.
